# SoomgoFlow

AI 기반 숨고 영업 자동화 플랫폼

## 기능

- 📊 칸반 보드 기반 요청 관리
- 🤖 AI 요청 분류 및 등급 매기기 
- ✍️ 자동 견적 초안 생성
- 🎯 드래그앤드롭으로 상태 관리
- 📝 상세 모달에서 견적 편집 및 승인

## 기술 스택

- **프론트엔드**: Next.js 14, TypeScript, Tailwind CSS
- **UI 컴포넌트**: shadcn/ui
- **상태 관리**: Zustand
- **드래그앤드롭**: dnd-kit
- **배포**: Vercel

## 시작하기

```bash
npm install
npm run dev
```

http://localhost:3000에서 확인할 수 있습니다.

## 데모 데이터

10건의 시드 데이터가 포함되어 있으며, 실제 숨고 크롤링 데이터를 기반으로 한 다음 등급 분류를 제공합니다:

- **A급** (1건): 김현 - Coze 워크플로우 API 연동
- **B급** (3건): 이수련, 김홍수, 정현태 - 웹 크롤링, VBA 개발 등
- **C급** (6건): 기타 요청들

## 칸반 보드 컬럼

1. **신규** - 새로 수집된 요청
2. **분류완료** - AI 분석 완료
3. **승인대기** - 견적 검토 대기
4. **발송완료** - 견적 발송됨
5. **진행중** - 프로젝트 진행 중
6. **완료** - 완료된 프로젝트
